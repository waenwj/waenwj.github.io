3. 性能优化技巧

PurgeCSS 配置

tailwind.config.js 配置：

module.exports = {
  content: [
    './src/**/*.{html,js,jsx,ts,tsx}',
    './public/**/*.html',
  ],
  purge: {
    enabled: process.env.NODE_ENV === 'production',
    content: [
      './src/**/*.{html,js,jsx,ts,tsx}',
      './public/**/*.html',
    ],
    options: {
      safelist: ['bg-red-500', 'text-blue-600']
    }
  }
}

条件类名优化

❌ 不推荐

<div class="bg-red-500 bg-blue-500">
  // 重复的类名
</div>

✅ 推荐

<div class="bg-red-500">
  // 单一职责
</div>

组件提取和复用

使用 @apply 指令提取组件（生产环境）：

/* 在 .css 文件中使用 @apply 指令 */
.btn-primary {
  @apply bg-blue-500 hover:bg-blue-600 text-white px-4 py-2 rounded transition-colors;
}

.btn-secondary {
  @apply bg-gray-500 hover:bg-gray-600 text-white px-4 py-2 rounded transition-colors;
}

.btn-danger {
  @apply bg-red-500 hover:bg-red-600 text-white px-4 py-2 rounded transition-colors;
}

.card {
  @apply bg-white rounded-lg shadow-md p-6 border border-gray-200;
}

.card-header {
  @apply border-b border-gray-200 pb-4 mb-4;
}

注意：@apply 指令需要在构建工具（如 PostCSS）环境中使用。在 CDN 环境中，我们使用原生 CSS 实现相同的效果。

4. 深色模式支持

💡 体验深色模式：点击右上角的切换按钮，体验整个页面的深色模式效果。所有带有 dark: 前缀的样式都会自动切换。

深色模式配置

tailwind.config.js：

module.exports = {
  darkMode: 'class', // 启用基于 class 的深色模式
  theme: {
    extend: {
      colors: {
        dark: {
          bg: '#1a202c',
          text: '#f7fafc',
        }
      }
    }
  }
}

注意：深色模式需要 JavaScript 来切换 dark 类到 html 元素上。

JavaScript 示例：

// 切换深色模式
function toggleDarkMode() {
  document.documentElement.classList.toggle('dark');
}

// 检查系统偏好
if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
  document.documentElement.classList.add('dark');
}

动画配置示例：

/* 自定义关键帧动画 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes slideUp {
  from { 
    transform: translateY(20px);
    opacity: 0;
  }
  to { 
    transform: translateY(0);
    opacity: 1;
  }
}

/* 应用动画 */
.animate-fade-in {
  animation: fadeIn 0.5s ease-in-out;
}

.animate-slide-up {
  animation: slideUp 0.5s ease-out;
}

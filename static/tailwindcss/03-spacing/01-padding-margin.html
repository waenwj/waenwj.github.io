间距系统 (Spacing System)

什么是间距系统？

Tailwind CSS 使用统一的间距比例系统，基于 0.25rem (4px) 的倍数。这使得设计更加一致和可预测。

间距比例：

0 0px
1 0.25rem (4px)
2 0.5rem (8px)
4 1rem (16px)
8 2rem (32px)
16 4rem (64px)

使用示例：

p-2 (padding: 0.5rem)
p-4 (padding: 1rem)
p-8 (padding: 2rem)

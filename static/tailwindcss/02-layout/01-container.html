容器 (Container)

什么是容器？

容器是一个用于限制内容最大宽度的工具类。它会在不同屏幕尺寸下自动调整最大宽度，并居中显示内容。

container mx-auto

这个容器会自动居中并限制最大宽度

移动端 (默认)

左右边距：1rem

最大宽度：1024px

使用的布局类：

  • • container mx-auto - 容器居中
  • • flex justify-between - 两端对齐
  • • items-center - 垂直居中
  • • space-x-4 - 水平间距
  • • hidden md:flex - 响应式显示

布局特点：

  • • 使用容器限制最大宽度
  • • Flexbox 实现水平布局
  • • 响应式设计适配移动端
  • • 合理的间距和对齐

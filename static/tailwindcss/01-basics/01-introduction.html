第1课：基础概念

什么是 Tailwind CSS？

传统 CSS 方式

传统方式需要写 CSS 类

需要定义 .button, .card 等类名

.button {
  background-color: #3b82f6;
  color: white;
  padding: 1rem;
  border-radius: 0.5rem;
  font-weight: bold;
}

Tailwind CSS 方式

直接使用工具类

bg-blue-500 text-white p-4 rounded font-bold

<button class="bg-blue-500 text-white p-4 rounded font-bold">
  按钮
</button>

实用优先 (Utility-First) 理念

实用优先意味着直接在 HTML 中使用预定义的工具类，而不是编写自定义 CSS。

🎨

快速开发

无需编写 CSS，直接在 HTML 中应用样式

📱

响应式设计

内置响应式工具类，轻松适配各种设备

🔧

一致性

统一的设计系统，保持界面一致性

实际示例：

卡片

使用的类：

w-16 h-16 bg-gradient-to-r from-blue-400 to-purple-500 rounded-lg flex items-center justify-center text-white font-bold

bg-red-500 text-white
bg-yellow-500 text-white
bg-purple-500 text-white

尺寸类 (Sizing)

h-8 (height: 2rem)
h-12 (height: 3rem)
h-16 (height: 4rem)
w-16 h-8
w-32 h-8

排版类 (Typography)

text-2xl font-semibold

text-lg text-gray-700

text-base text-gray-600

text-sm text-gray-500
